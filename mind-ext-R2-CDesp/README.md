# R2 Extension


![](./arduinoC/_images/featured.png)

---------------------------------------------------------


## Summary
Mind+ extension for R2 robot by Polytech.
Supports text LCD 16x2 or 20x4 via I2C 
Supports arbitrary degrees turn utilizing a QMC8533 Magnetometer via I2C
